Order Brother!
===

## Requirements

* Ruby version 2.6.0
* Rails versino 5.2.3
* PostgreSQL 11.3

## Usage

### Install Dependency

```
$ bundle install
```

### Prepare Database

If you didn't have database.

```
$ rake db:create
```

And run migration to create tables.

```
$ rake db:migrate
```

### Rails

Start Rails server

```
$ rails s
```
